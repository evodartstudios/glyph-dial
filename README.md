> **Nothing Dialer** is EvoDART's first official open-source project — a community-driven, fully functional replacement for the stock Android phone app, built entirely with Jetpack Compose.

---

## ✨ Features

| Category | Feature |
|---|---|
| **Dialing** | T9 predictive search (Trie-based), DTMF in-call tones, voicemail speed-dial (long-press 1) |
| **Calls** | Full incoming/outgoing/hold/conference lifecycle, call waiting banner, reject-with-SMS |
| **Notifications** | Live call duration timer, Mute/Speaker/Merge quick actions, missed call |
| **Contacts** | Search, alphabet fast-scroll, favorites grid, contact detail with full call history |
| **Call Log** | Stacked grouped entries, swipe-to-delete, Clear All, add-unknown-to-contacts chip |
| **Audio** | Dynamic route picker (Earpiece / Speaker / Bluetooth), proximity sensor auto screen-off |
| **SIM** | Dual-SIM detection & routing, default SIM preference persisted in settings |
| **Emergency** | All emergency numbers (112, 911, 999, 110…) bypass the app and route via system dialer |
| **UI** | Nothing-inspired dark theme, custom accent colors (8), dot-matrix typography, smooth animations |
| **Settings** | Default dialer setup, accent color, start page, scrollbar position, calls sub-screen |

---

## 🛠 Requirements

- **Android Studio Hedgehog** or newer
- **Minimum SDK:** API 28 (Android 9.0)
- **Target SDK:** API 35 (Android 15)
- A **physical device** is strongly recommended for testing call functionality

---

## 🚀 Getting Started

```bash
# 1. Clone
git clone https://github.com/evodart/Nothing-Dialer.git

# 2. Open in Android Studio → File → Open
# 3. Sync Gradle
# 4. Run on device (call features require a physical SIM)
```

> ⚠️ The app needs to be set as **Default Phone App** to handle calls. You'll be prompted on first launch.
