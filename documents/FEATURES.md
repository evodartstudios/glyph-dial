# Glyph Dial - Complete Features List

> A next-generation dialer app for Nothing Phone with distinctive Nothing OS design language

## 📞 Core Dialer Features

### Dial Pad
- **Dot Matrix Dial Pad** - Nothing-style dot matrix number buttons with haptic feedback
- **T9 Smart Search** - Search contacts while typing numbers
- **Quick Actions** - Long-press for speed dial assignment
- **Number Formatting** - Auto-format as you type with country detection
- **Copy/Paste Support** - Smart number extraction from clipboard
- **Emergency Call Quick Access** - Swipe up for emergency dial

### Call Management
- **Incoming Call Screen** - Full-screen Nothing-style UI with Glyph animations
- **Outgoing Call Screen** - Minimalist call progress with dot matrix timer
- **In-Call Controls** - Speaker, mute, hold, keypad, add call, record
- **Call Waiting** - Seamless call switching with visual indicators
- **Conference Calling** - Merge calls with participant management
- **HD Voice Indicator** - Visual indicator for VoLTE/VoWiFi calls
- **Call Timer** - Dot matrix style real-time call duration

### Call History
- **Timeline View** - Chronological call log with day groupings
- **Call Type Filters** - All, missed, incoming, outgoing, blocked
- **Smart Grouping** - Consecutive calls to same contact grouped
- **Quick Actions** - Swipe to call back, message, or block
- **Call Details** - Duration, time, SIM used, call quality metrics
- **Search** - Full-text search across call history
- **Export/Import** - Backup and restore call logs
